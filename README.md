# Cloudinary Media Management Service

A Spring Boot service that simplifies Cloudinary integration for media file management. Upload, organize, and delete images, videos, and raw files with ease.

## ✨ Features

- **📸 Image Upload** - Optimized image upload with automatic format detection
- **🎥 Video Upload** - Seamless video file upload to Cloudinary
- **📄 Raw File Upload** - Support for PDFs, documents, and other file types
- **🗑️ File Deletion** - Clean removal of files with cache invalidation
- **📁 Folder Organization** - Automatic file organization in Cloudinary folders
- **🔒 Secure URLs** - HTTPS secure URL generation for all uploaded files
- **⚡ Fast Integration** - Ready-to-use REST endpoints
